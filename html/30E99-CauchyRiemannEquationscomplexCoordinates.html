Cauchy-Riemann equations (complex coordinates)


Let f:G⊂ℂ→ℂ be a continuously differentiable function in the real sense, using ℝ2 instead of ℂ, identifying f⁢(z) with f⁢(x,y) where z=x+i⁢y and we also write z¯=x-i⁢y (the complex conjugate). Then we have the following partial derivatives:

∂⁡f∂⁡z :=12⁢(∂⁡f∂⁡x-i⁢∂⁡f∂⁡y),
∂⁡f∂⁡z¯ :=12⁢(∂⁡f∂⁡x+i⁢∂⁡f∂⁡y).

Sometimes these are written as fz and fz¯ respectively.

The classical Cauchy-Riemann equations are equivalent to

∂⁡f∂⁡z¯=0.

This can be seen if we write f=u+i⁢v for real valued u and v and then the differentials become

∂⁡f∂⁡z =12⁢(∂⁡u∂⁡x+∂⁡v∂⁡y)+i2⁢(∂⁡v∂⁡x-∂⁡u∂⁡y),
∂⁡f∂⁡z¯ =12⁢(∂⁡u∂⁡x-∂⁡v∂⁡y)+i2⁢(∂⁡v∂⁡x+∂⁡u∂⁡y).

In several complex dimensions, for a function f:G⊂ℂn→ℂ which maps (z1,…,zn)↦f⁢(z1,…,zn) where zj=xj+i⁢yj we generalize simply by

∂⁡f∂⁡zj :=12⁢(∂⁡f∂⁡xj-i⁢∂⁡f∂⁡yj),
∂⁡f∂⁡z¯j :=12⁢(∂⁡f∂⁡xj+i⁢∂⁡f∂⁡yj).

Then the Cauchy-Riemann equations are given by

∂⁡f∂⁡z¯j=0  for all 1≤j≤n.

That is, f is holomorphic if and only if it satisfies the above equations.
