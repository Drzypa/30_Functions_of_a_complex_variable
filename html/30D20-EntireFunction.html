entire function


An entire function is a function f:ℂ⟶ℂ which is holomorphic everywhere on the complex domain ℂ.

For example, a polynomial is holomorphic everywhere, as is the exponential function. The function z↦1/z is not holomorphic at zero, so it is not entire; it is meromorphic.
