values of gamma function for small positive real values


The following table lists values of Γ⁢(x) for real 0≤x<10 in steps of 110. If the table is correct, it should confirm for a real nonnegative integer n the following equality: Γ⁢(n)=(n-1)!, with the notable exception of n=0, for which the value is complex infinity. Generally speaking, for values of x in the relation n<x<(n+1), the relation Γ⁢(n)<Γ⁢(x)<Γ⁢(n+1) always holds, but not when 0<x<2.
