(ii) Now assume ℱ to be normal. Define:

f♯⁢(z) :=2⁢|f′⁢(z)|1+|f⁢(z)|2

Let K⊆Ω be compact. To obtain contradiction assume {f♯:f∈ℱ} is not uniformly bounded on K. But then there exists a sequence {fn}⊂ℱ such that:

maxz∈K⁡fn♯⁢(z) →∞(n→∞)

Since ℱ is normal for each P∈Ω let there be a neighbourhood UP such that {fn} converges normally to a meromorphic function f. But from (1/fn)♯=fn♯ we see that {fn♯} converges normally on UP. Since K is compact it can be covered by finitely many sets UP. We conclude that {fn♯} must be bounded on K and obtain a contradiction. ∎
