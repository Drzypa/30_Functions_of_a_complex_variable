proof of Gauss’ mean value theorem


We can parameterize the circle by letting z=z0+r⁢ei⁢ϕ. Then d⁢z=i⁢r⁢ei⁢ϕ⁢d⁢ϕ. Using the Cauchy integral formula we can express f⁢(z0) in the following way:

f⁢(z0) = 12⁢π⁢i⁢∮Cf⁢(z)z-z0⁢𝑑z
= 12⁢π⁢i⁢∫02⁢πf⁢(z0+r⁢ei⁢ϕ)r⁢ei⁢ϕ⁢i⁢r⁢ei⁢ϕ⁢𝑑ϕ
= 12⁢π⁢∫02⁢πf⁢(z0+r⁢ei⁢ϕ)⁢𝑑ϕ.
